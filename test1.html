🔒 Permission Denied — Fix Your Firestore Rules

Go to Firebase Console → Firestore Database → Rules and set:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

⚠️ This opens full access for testing only. Add proper auth rules before going live.
